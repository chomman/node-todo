# Node TODO application

[NestJS Framework](https://github.com/nestjs/nest)을 사용한 todo 앱 구현 예시

## 기술 스택
- node.js
- typescript
- nest.js
- typeorm

## 라이브러리 설치

```bash
$ npm install
```

## 앱 실행

```bash
$ npm run start
```

http://localhost:8080/api-docs/ 에서 api document 확인


## Debug

```bash
$ npm run start:debug
```

## Test

```bash
$ npm run test:e2e
```
